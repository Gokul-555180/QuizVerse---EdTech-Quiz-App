# 🚀 QuizVerse

> **A modern, interactive quiz application built with React and Vite, featuring a premium glassmorphism interface, smooth animations, and a responsive user experience.**

QuizVerse is a frontend-focused React project that combines modern UI/UX design with an engaging quiz experience. The project emphasizes clean architecture, responsive layouts, smooth animations, and a polished design inspired by modern SaaS applications.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/quizverse.git
```

Go to the project directory

```bash
cd quizverse
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
